# Pheme Hermes — Social Media Manager Worker

HAMH Free Tier default worker.

## Role
Content creation, post scheduling, engagement, analytics, and content strategy across social platforms.

## Capabilities
- `post-x` — Post to X/Twitter
- `engagement` — Reply, like, retweet, follow
- `analytics` — Metrics and reporting
- `schedule` — Content calendar and post scheduling
- `content-strategy` — Campaign planning and trend analysis
- `curate-content` — RSS monitoring and source aggregation

## Setup
```bash
cp .env.example .env
npm install
npm start
```
