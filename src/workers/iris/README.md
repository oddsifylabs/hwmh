# Iris Hermes — Admin Assistant Worker

HAMH Free Tier default worker.

## Role
Calendar scheduling, email drafting, meeting notes, file organization, research, and documentation.

## Capabilities
- `admin` — General administrative tasks
- `schedule` — Calendar events and scheduling
- `research` — Information gathering and fact-checking
- `documentation` — Notes, wikis, markdown generation
- `email` — Drafting and inbox triage
- `reminder` — Notifications and follow-ups

## Setup
```bash
cp .env.example .env
npm install
npm start
```
