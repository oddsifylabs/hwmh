# Kairos Hermes — Sales & Marketing Worker

HAMH Free Tier default worker.

## Role
Lead generation, outreach, CRM management, campaign creation, and sales pipeline tasks.

## Capabilities
- `lead-generation` — Prospecting and list building
- `sales` — Outreach sequences and proposals
- `marketing` — Campaigns, ad copy, funnel analytics
- `crm` — HubSpot, Salesforce, or Airtable integration
- `analytics` — Pipeline metrics and conversion tracking
- `content-strategy` — SEO research and competitor analysis

## Setup
```bash
cp .env.example .env
npm install
npm start
```
